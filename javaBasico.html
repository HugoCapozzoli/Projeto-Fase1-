POO EM JAVA

O que é Programação Orientada a Objetos
- A Programação Orientada a Objetos (POO) é um estilo de programação onde os conceitos do mundo real são modelados como "objetos" que possuem características (atributos) e comportamentos (métodos), permitindo uma organização modular e reutilizável do código. Esses objetos interagem entre si, facilitando o desenvolvimento de software mais eficiente e fácil de entender.

JDK (Java Development Kit)
- NetBeans é um ambiente de desenvolvimento integrado (IDE)

O que é um Objeto?
- Nessa aula de POO, vamos aprender os conceitos de Classes e Objetos, passando pela teoria de Atributos, Métodos, Estado e Instâncias. Veja como criar uma classe e instanciar, criando objetos.

Criando Classes e Objetos em Java
-Nessa aula de POO, vamos aprender como criar uma Classe, com seus atributos e métodos, e a instanciar novos objetos a partir dessa classe inicial.

O que é Visibilidade em um Objeto?
- Nessa aula de POO, vamos aprender qual a importância dos modificadores de visibilidade público (+), privado (-) e protegido (#) na Programação Orientada a Objetos.

Configurando Visibilidade de Atributos e Métodos
- Nessa aula de POO, vamos aprender na prática como usar as palavras public, private e protected na linguagem Java e qual é o efeito no uso de cada uma delas.

Métodos Especiais
- Nessa aula de POO, vamos aprender como funcionam os Métodos Acessores (Getters), Métodos Modificadores (Setters) e Métodos Construtores (Construct) para a Programação Orientada a Objetos.

Métodos Getter, Setter e Construtor
- Nessa aula de POO, vamos aprender como criar em Java os Métodos Acessores (Getters), Métodos Modificadores (Setters) e Métodos Construtores (Construct).

Exemplo Prático com Objetos
- Nessa aula de POO, vamos fazer um exemplo prático com Programação Orientada a Objetos, usando tudo aquilo que aprendemos até aqui.

Exemplo Prático em Java
